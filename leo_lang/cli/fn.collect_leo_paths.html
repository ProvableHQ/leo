pub fn collect_leo_paths(package: &Package) -> Vec<(PathBuf, Vec<PathBuf>)>
Expand description

Collects paths to Leo source files for each program in the package.

For each non-test program, it searches the src directory for .leo files. It separates the main.leo file from the rest and returns a tuple: (main.leo path, list of other .leo file paths). Test programs are included with an empty list of additional files. Programs with bytecode data are ignored.
