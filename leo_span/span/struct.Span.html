Struct leo_span::span::Span

source · 
pub struct Span {
    pub lo: BytePos,
    pub hi: BytePos,
}
Expand description

The span type which tracks where formatted errors originate from in a Leo file. This is used in many spots throughout the rest of the Leo crates.

Fields§

§lo: BytePos

The start (low) position of the span, inclusive.

§hi: BytePos

The end (high) position of the span, exclusive. The length of the span is hi - lo.
