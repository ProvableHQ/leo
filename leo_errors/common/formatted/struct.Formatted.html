pub struct Formatted {
    pub span: Span,
    pub backtrace: Backtraced,
}
Expand description

Formatted compiler error type undefined value x –> file.leo: 2:8 | 2 | let a = x; | ^ | = help: Initialize a variable x first. Makes use of the same fields as a BacktracedError.
